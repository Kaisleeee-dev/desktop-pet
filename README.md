# 🎀 Desktop Pet - 二次元桌面宠物

一款可爱的 2D 二次元风格桌面宠物应用，基于 Electron + React 开发。

## ✨ 功能特性

- 🎨 **精美的二次元风格宠物设计** - 拟人化的粉色萌宠
- 🖱️ **鼠标跟踪** - 宠物会跟随你的鼠标移动
- 💬 **丰富的交互** - 点击宠物获得反应
- 📊 **完整的属性系统**：
  - 🍔 饥饿度
  - 😊 快乐度
  - ⚡ 能量值
  - ❤️ 健康度
- 🎮 **互动菜单** - 喂食、玩耍、休息、洗澡
- 🎭 **多种动画状态** - 闲置、走路、开心、疲劳、难过
- 💾 **状态实时显示** - 侧边栏显示宠物各项属性
- 🎵 **现代化界面** - 毛玻璃效果和渐变设计

## 🚀 快速开始

### 前置要求

- Node.js >= 14.0
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm start
```

这将同时启动 React 开发服务器和 Electron 应用。

### 构建应用

```bash
npm run build
```

生成的应用将在 `dist` 文件夹中。

## 🎮 操作说明

| 操作 | 说明 |
|------|------|
| 移动鼠标 | 宠物跟随鼠标 |
| 点击宠物 | 宠物做出开心表现 |
| 点击右上角⚙️ | 打开菜单 |
| 🍕 喂食 | 降低饥饿度，增加健康度 |
| 🎮 玩耍 | 增加快乐度，降低能量值 |
| 😴 休息 | 恢复能量值 |
| 🛁 洗澡 | 增加健康度 |

## 🎨 自定义宠物

您可以编辑 `src/components/Pet.css` 来自定义宠物的外观：

- 修改颜色渐变
- 调整大小和比例
- 改变动画速度
- 添加新的部位设计

## 🔧 配置选项

在 `src/App.js` 中的 `petState` 对象中修改初始属性值：

```javascript
const [petState, setPetState] = useState({
  hunger: 50,      // 0-100
  happiness: 70,   // 0-100
  energy: 80,      // 0-100
  health: 90,      // 0-100
  // ...
});
```
